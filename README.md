# homebrew-tap

Homebrew tap for [Stavros](https://github.com/skorokithakis)' tools.

## Usage

```sh
brew tap skorokithakis/tap
brew install --cask <name>
```

## Available packages

- **[gnosis](https://github.com/skorokithakis/gnosis)** — `brew install --cask skorokithakis/tap/gnosis`. A knowledge base for what the code can't tell you.
